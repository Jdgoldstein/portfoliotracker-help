Setup steps — do these once, in order
1
Create a permanent working folder and move the file in
Create a dedicated folder somewhere permanent — for example C:\PortfolioTracker\ on Windows or Documents/PortfolioTracker/ on Mac. Move Portfolio_Tracker_Demo.html into it (out of your Downloads folder).
Why permanent? In Step 3 you link the tracker to a data file that sits next to this HTML. If you later move or delete the folder, just re-link — your data is never lost, only the link.
2
Tell Chrome to ask where to save files Important
By default Chrome saves files to Downloads — which means the tracker's data file ends up in the wrong place.
  1. In Chrome paste chrome://settings/downloads into the address bar and press Enter
    (Edge: edge://settings/downloads)
  2. Turn ON — "Ask where to save each file before downloading"
  3. Close Settings
Why this matters: When Chrome asks where to save, you point your data file at your working folder in Step 3. Your other downloads are unaffected.
3
Open the tracker and link your data file Most important
Double-click Portfolio_Tracker_Demo.html from your working folder. Not from Downloads.
  1. Click ▶ Start Free Trial
  2. Click ⛁ Data ▾ → Link / Change File
  3. Navigate to your working folder
  4. Type portfolio_data.json and click Save
  5. Status dot turns green ● — auto-save active
Green dot = protected. Saves every 7 minutes automatically.
If the dot turns grey: click ⛁ Data ▾ → Link / Change File and re-select your portfolio_data.json. Data is safe — just re-link.
4
Create a desktop shortcut
Windows: Right-click Portfolio_Tracker_Demo.html in Explorer → Send to → Desktop (create shortcut).

Mac: Hold Cmd + Opt and drag the HTML file to the Desktop.
Optional — give your shortcut the tracker icon (Windows) Download Portfolio_Tracker.ico using the button, then right-click your shortcut → Properties → Change Icon… → Browse… → select it → OK → Apply.

Optional — install as a desktop app
⊕
Install as a PWA Optional
Chrome and Edge can install the tracker as a standalone app with its own window and taskbar icon. Most users don't need this.

With the tracker open: click ⊕ Install App in the top-right header, or use Chrome's menu (⋮) → Save and share → Install page as app. Once installed it appears in your Start Menu with the correct icon automatically.
🍎 Mac users — one extra step
!
Allow Chrome to save files automatically Mac only
The first time the tracker auto-saves, Chrome asks "Allow this site to download multiple files?" — click Allow. Only appears once.

Restoring after a cache clear: Click ⛁ Data ▾ → Import Backup, select portfolio_data.json, then re-link via Link / Change File.
Good to know
💡
Your data lives in two places
Safe copy: portfolio_data.json — saved every 7 minutes automatically.

Temporary: Browser localStorage — wiped if you clear cache. Always keep the .json file safe.

Use ⛁ Data ▾ → Save / Export Backup occasionally for an extra dated copy.
💡
Updating to a new version
Download the new Portfolio_Tracker_Demo.html and drop it into your working folder — overwriting the old one. Do not replace your portfolio_data.json unless specifically told to. Your data stays intact.
💡
Use Chrome or Edge — not Firefox or Safari
The File System Access API used for auto-save is not supported in Firefox or Safari.
