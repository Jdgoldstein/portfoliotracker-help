Welcome to Portfolio Tracker

A self-contained crypto portfolio tracker that runs entirely in your browser — no installation, no account, no cloud. Follow the four steps below in order.

✓ Works in Chrome or Edge  ·  No installation needed  ·  Your data never leaves your machine
⚠ Before anything else — save the file to your computer

You are viewing this from a web link. The tracker must be saved to a folder on your computer first — do not just click Open or it will run from your Downloads folder and data won't persist reliably.

Right-click the button below → "Save link as…" and save Portfolio_Tracker_Demo.html to your computer. Then follow the four steps below.

Four steps — do these once, in order
1
Create a permanent working folder
Create a dedicated folder somewhere permanent — for example C:\PortfolioTracker\ on Windows or Documents/PortfolioTracker/ on Mac.

Move both files into this folder: Portfolio_Tracker_Demo.html and this START_HERE.html. Keep everything here — don't move it later.
2
Point Chrome's download location at your working folder Do this first
The tracker saves your data by writing a .json file to disk. By default Chrome saves files to your Downloads folder — which means your data ends up in the wrong place and the tracker loses the link every time Chrome restarts.

Fix this before opening the tracker:
  1. In Chrome, paste chrome://settings/downloads into the address bar and press Enter
    (Edge users: use edge://settings/downloads)
  2. Next to Location, click Change…
  3. Navigate to your working folder and click Select Folder
  4. Make sure "Ask where to save each file" is turned OFF
Why this matters: Once set, every auto-save lands silently in your working folder as portfolio_data.json — right next to the tracker HTML file. The link is stable across restarts. Without this step the file drifts to Downloads and the tracker shows a warning dot every time you open it.
3
Open the tracker and link your data file Most important step
Double-click Portfolio_Tracker_Demo.html from your working folder to open it in Chrome or Edge. Not from Downloads — from your working folder.
  1. Click ▶ Start Free Trial on the activation screen
  2. In the header, click ⛁ Data ▾ → Link / Change File
  3. A save dialog opens — you should already be in your working folder (Step 2 ensures this). If not, navigate there manually.
  4. Type portfolio_data.json as the filename and click Save
  5. The status dot in the header turns green ● — auto-save is now active
Green dot = you're protected. The tracker now saves automatically every 7 minutes and whenever you switch away from the tab. Your data file portfolio_data.json sits in your working folder — you can see it in Explorer or Finder.
If the dot ever turns grey or orange: the file link was lost. Click ⛁ Data ▾ → Link / Change File, navigate to your working folder, and select the existing portfolio_data.json. Your data is safe in the file — just re-link it.
4
Create a desktop shortcut
Windows: Open your working folder in Explorer. Right-click Portfolio_Tracker_Demo.html → Send to → Desktop (create shortcut).

Mac: Open Finder and navigate to your working folder. Hold Cmd + Opt and drag the HTML file to the Desktop — this creates an alias without moving the original.

Always open the tracker via this shortcut or directly from the working folder — never from a browser bookmark to a downloaded file.
Optional — add the Portfolio Tracker icon to your shortcut (Windows only) Download the icon and save it to your working folder. Then right-click your desktop shortcut → Properties → Change Icon… → Browse… → select Portfolio_Tracker.ico → OK → Apply.
The browser tab icon is set automatically — this step only affects the desktop shortcut.

Optional — install as a desktop app (advanced)
⊕
Install as a PWA desktop app Optional
Chrome and Edge can install the tracker as a standalone desktop app with its own window and taskbar icon — no browser tabs or address bar. Most users don't need this — the browser tab version works identically.

If you want to try it: with the tracker open in Chrome, click the ⊕ Install App button in the tracker's top-right header, or use Chrome's menu (⋮) → Save and share → Install page as app.

Once installed, the app appears in your Start Menu with the correct icon automatically. The data file link set up in Step 3 carries over to the installed app.
Note: If you use the PWA version, you can skip the desktop shortcut and icon steps above — the installed app handles all of that automatically.
🍎 Mac users — one extra step
!
Allow Chrome to save files automatically Mac only
The first time the tracker auto-saves, Chrome shows a bar asking "Allow this site to download multiple files?" — click Allow. This only appears once.

If you missed it or clicked Block: go to Chrome → Settings → Privacy and security → Site settings → Additional permissions → Automatic downloads and add the tracker to the Allowed list.

Restoring after a cache clear: Click ⛁ Data ▾ → Import Backup, select portfolio_data.json from your working folder, then re-link it via Link / Change File.
Good to know
💡
Your data lives in two places — know the difference
Safe copy: portfolio_data.json in your working folder — saved automatically every 7 minutes. This is your real data.

Temporary copy: Browser localStorage — fast but wiped if you clear your browser cache. Always keep the .json file safe.

For extra peace of mind, use ⛁ Data ▾ → Save / Export Backup occasionally to download a dated copy somewhere else.
💡
Use Chrome or Edge — not Firefox or Safari
The tracker uses the File System Access API to save your data automatically. This API is not supported in Firefox or Safari — auto-save won't work in those browsers.
